Stepik Auto Tests Course
This repository contains my homework assignments for the Automation Testing with Python course on Stepik.

The course covers writing automated UI tests using Selenium WebDriver and Python, including:

Setting up Selenium and ChromeDriver
Finding elements with CSS, XPath, and other locators
Handling checkboxes, radio buttons, dropdowns, and file uploads
Managing alerts, windows, and tabs
Using explicit and implicit waits
Writing stable tests with unique selectors
Basics of Git and version control
